Inventory Management System API

A RESTful Inventory and Order Management System built using Django REST Framework with JWT Authentication, Stock Tracking, Dashboard APIs, Search, Ordering, Pagination, and Swagger Documentation.

Features
* JWT Authentication
* Category Management
* Supplier Management
* Product Management
* Customer Management
* Order Management
* Order Item Management
* Stock Validation
* Automatic Stock Reduction
* Low Stock API
* Dashboard API
* Search, Ordering, Pagination
* Swagger Documentation

Technologies Used
* Python
* Django
* Django REST Framework
* SQLite
* JWT Authentication
* Swagger (drf-yasg)
* Git & GitHub
